Project Title: Assistive Device Annotation for Healthcare AI

Description:
This project contains annotated rehabilitation images to identify walking assistive devices.

Labels Used:
- walker
- crutch
- cane
- no device

Annotation Tool:
makesense.ai

Annotation Method:
Bounding boxes were drawn around assistive devices.
For no-device cases, the bounding box was drawn around the person.

Output Format:
YOLO

Use Case:
This dataset can be used for AI-based mobility assessment,
fall-risk analysis, and rehabilitation monitoring.
